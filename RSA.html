RSA Response 3a:

The program takes an input or message from the user and then encrypts it using the public keys, e and n, resulting in the output. After encrypting the message, the program can decrypt the output using the private keys, n and d. The functionality of the program that is demonstrated is the user typing their message and then after they enter values for the public keys, e and n, the program encrypts the message. The same occurs in the decryption process which occurs after the user saves and opens the file with their encrypted message. After entering values for the private keys, d and n, the program decrypts the encrypted message. The input was the user’s message along with the values for the public key. Once the message was encrypted, the user entered values for the private key and then clicked the “decrypt” button which resulted in the decrypted output.
